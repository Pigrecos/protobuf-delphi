en New features

Version 1.1
1. Code debugged and compiled using a version of Embarcadero XE3.
2. For strings used AnsiString.
3. Added an example of the use and the unit tests for this example.
4. Easier to work with nested messages.

Version 1.11
1. Fixed bug with empty string
2. Added a unit test for an empty string
3. The code formatted according to the Delphi coding standard
4. For compiling in Delphi 10.2, there were no tests for ZagZig Int64. Fixed.
